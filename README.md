# Part 1 ‚Äì Personality Quiz

## Base Implementation
This project is based on Apple's "Personality Quiz" guided project from the Develop with Swift Fundamentals book (pages 533‚Äì600).  
The full base functionality has been implemented using UIKit and Storyboards.

---

## ‚úÖ Stretch Goals Implemented

1. *Multiple Quiz Selection*
Users can choose from multiple personality quizzes via the intro screen.

2. *Randomized Questions and Answers*
Both questions and answers are shuffled every time the quiz starts.

3. *Dynamic Answer Support*
Answer buttons are generated dynamically using UIStackView and support both single and multiple-answer questions.

4. *Quiz History Screen*
Completed quiz results are stored locally and displayed in a dedicated history screen.

5. *Quiz Timer*
Each question includes a countdown timer limiting the time available to answer.
